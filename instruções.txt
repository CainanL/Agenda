---------django-----------
starta a plicação -> python manage.py runserver 
cria um novo app -> python manage.py startapp nomedoapp

---------banco de dados-----------
cria uma migração a partir do modelo -> python manage.py makemigrations
aplicar migrations -> python manage.py migrate

--------admin-------------
cria um usuário -> python manage.py createsuperuser

lib para trabalhar com imagens
